Capability Registry
Enforces domain invariants: validates Safety Envelope and Risk Tier at publish (I2), blocks execution of unpublished Definitions (I1), rejects R2/R3 without Workflow Contract (I4). System of record — versioned, SHA-256 hashed. Runtime ABAC-filtered tool discovery via Registry API

Cost & Budget Control
Real-time spend tracking via LiteLLM virtual keys — one key per tenant with configurable monthly budget cap. Circuit breaker cuts traffic at 100% budget consumed. Cost attributed per tenant / agent / session across Bedrock, Azure OpenAI, and Gemini routes

Audit & Compliance
Writes the Audit Record (I5) — immutable, synchronous on every Execution. Identity Context, Safety Envelope version, Risk Tier in effect, all MCP tool calls and results, guardrail traces, token counts, cost. SHA-256 hashed against Registry version entry. S3 Object Lock retention driven by Risk Tier. Fail-closed for R2/R3 — no state-changing tool calls without audit commit
S3 Object LockKMS per tenant

Implements the Workflow Contract for R2/R3 Capability Definitions. Step Functions state machines provide durable execution, deterministic sequencing, retries, timeouts, parallel branches, and compensation. ABAC enforced at each step boundary via Identity Context
