Validates identity on every API Gateway request. Calls STS GetCallerIdentity to verify the IAM session established by IAM Identity Center, then extracts session tags (tenant_id, brand, role, use-case). Establishes the Identity Context for the Execution. Result cached 5 min per session.
→ IAM STS GetCallerIdentity · verify session · extract session tags AWS SDK · sts.getCallerIdentity
→ IAM STS AssumeRole with session tags (tenant_id · brand · role · use-case) for downstream ABAC AWS SDK · sts.assumeRole

Runtime Orchestration
Agent Orchestration Service ECS Fargate · Python · Strands SDK
Thin application wrapper around AgentCore / Strands. Loads Published Capability Definition from Registry. Constructs the agent session with Identity Context. Passes control to Strands for the reasoning loop. Writes Audit Record on close.
→ Registry Service Resolve Published Capability Definition by ID + version internal REST GET
→ AgentCore / Strands Delegate agent loop execution — tools, model calls, state Strands SDK · Python
→ DynamoDB Session state reads/writes · conversation history boto3 · get/put_item
→ S3 (Audit) Synchronous Audit Record write on Execution close (I5) boto3 · put_object · Object Lock
→ Step Functions Start Process-scope execution for R2/R3 capabilities boto3 · sfn.start_execution

MCP Gateway Service ECS Fargate · Python
Outbound tool call router. Receives tool invocation from Strands agent loop. Performs RFC 8693 token exchange to produce a tool-scoped credential carrying Identity Context. Routes to the target MCP server. ABAC validated against session tags before dispatch.
→ IAM STS RFC 8693 token exchange — session credential → tool-scoped credential boto3 · sts.assume_role_with_web_identity
→ Secrets Manager Fetch MCP server credentials / API keys per tool target boto3 · get_secret_value
→ Downstream MCP servers HTTP tool invocation under scoped credential — CRM · RG · ITSM · etc. MCP HTTP transport · mTLS
no direct agent egressRFC 8693
HITL Service ECS Fargate · Python
Human-in-the-loop approval gate. Receives Step Functions waitForTaskToken callback. Routes approval request to reviewer via SQS queue. Records reviewer identity, decision, and timestamp. Sends task token success/failure back to Step Functions. Non-waivable for R3.
← Step Functions waitForTaskToken activity — blocks execution until decision sfn activity / task token
→ SQS Approval request enqueued for reviewer notification boto3 · sqs.send_message
→ Step Functions send_task_success / send_task_failure with decision payload boto3 · sfn.send_task_success
R3 mandatoryECS
Safety & Model Access
LLM Gateway ECS Fargate · LiteLLM · Python
LiteLLM-based proxy between agent runtime and model providers. Enforces per-tenant quota (burst + sustained window). Manages prompt cache keys. Routes to Bedrock, Anthropic direct, OpenAI, or Google based on Capability Definition declaration. Opens circuit breaker on budget cap breach. Attributes cost per tenant/capability in real time.
→ Bedrock Runtime InvokeModel / InvokeModelWithResponseStream · Claude · Titan · Llama boto3 · bedrock-runtime
→ Bedrock Guardrails ApplyGuardrail on every prompt + response before/after model call boto3 · bedrock.apply_guardrail
→ ElastiCache (Redis) Quota counter reads/writes · burst window · prompt cache key store redis-py · INCR / EXPIRE
→ Langfuse LiteLLM callback — emits span per model call with tokens, latency, cost LiteLLM callback hook · HTTP
→ CloudWatch Per-tenant cost metrics · quota utilisation · circuit breaker state boto3 · put_metric_data

Audit & Observability Writes
Audit Record Writer Library · shared by all services
Shared library embedded in the Agent Orchestration Service and Step Functions state handlers. Synchronous write — not fire-and-forget. For R2/R3 the calling service blocks until the S3 put_object call returns successfully. For R3 called before any state-changing tool invocation (write-ahead).
→ S3 (Audit) put_object with Object Lock retention — Governance mode R0/R1 · Compliance R2/R3 boto3 · put_object · object-lock-mode
→ KMS Encrypt Audit Record with account CMK before write KMS · server-side encryption
